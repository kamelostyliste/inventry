# MySales - نظام إدارة المبيعات والمخزون

تطبيق ويب ثنائي اللغة (عربي/إنجليزي) لإدارة المبيعات والمخزون مع تقارير متقدمة.

## التقنيات

- **React 19** + **Vite 8**
- **Supabase** (قاعدة بيانات + مصادقة)
- **Recharts** (رسوم بيانية)
- **Lucide React** (أيقونات)
- **React Router 7**

## التشغيل

```bash
npm install
npm run dev
```

## البناء للإنتاج

```bash
npm run build
```

## متغيرات البيئة

انسخ ملف `.env.example` إلى `.env` واملأ القيم:

```
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

إذا لم يتم ضبط المتغيرات، يعمل التطبيق في وضع **offline** باستخدام `localStorage`.

## المميزات

- لوحة تحكم مع إحصائيات ورسوم بيانية
- إدارة المنتجات (مع رفع الصور وتاريخ الحركات)
- إدارة العملاء (كشوف حساب)
- إدارة الطلبيات (فاتورة، حالة الدفع، التوصيل)
- تقارير متقدمة مع تصفية زمنية
- دعم كامل للغة العربية (RTL) والإنجليزية
- وضع مظلم/فاتح
- استيراد/تصدير CSV
- طباعة PDF للتقارير والفواتير

## هيكل المجلدات

```
src/
  components/   # مكونات قابلة لإعادة الاستخدام
  context/      # مزودات الحالة (Auth, Data, Notification)
  pages/        # صفحات التطبيق
  services/     # خدمات Supabase
  utils/        # أدوات مساعدة
```
